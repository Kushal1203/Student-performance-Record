
# Student Performance Analyzer

A comprehensive application designed to track, analyze, and manage student academic performance for Sha-Shib College of Technology. This tool simplifies the process of recording marks, calculating grades (SGPA/CGPA), and generating performance reports.

## üöÄ Features

-   **User Authentication**: Secure Login and Signup functionality for authorized access.
-   **Dashboard**: Interactive dashboard to view student statistics and performance metrics.
-   **Student Management**: Add, update, delete, and view detailed student profiles.
-   **Academic Records**: Track marks for various subjects across semesters.
-   **Automated Calculations**: Automatically calculates Grades, SGPA, and CGPA based on entered marks.
-   **Subject Management**: Manage subjects and their credit values for each semester.
-   **Batch & Department Filtering**: Filter data by specific batches and academic departments.
-   **Library & Accounts**: Basic tracking for library usage and fee status.
-   **Responsive Design**: Modern UI built with Tailwind CSS for a seamless experience on different devices.

## ‚öôÔ∏è Installation & Running

### Prerequisites
-   Node.js installed
-   Python installed

### 1. Backend Setup
Navigate to the backend directory and start the server:
```bash
cd python_backend
pip install -r requirements.txt
python main.py
```
The backend server will start at `http://localhost:3001`.

### 2. Frontend Setup
In a new terminal, navigate to the root directory and start the React application:
```bash
npm install
npm run dev
```
